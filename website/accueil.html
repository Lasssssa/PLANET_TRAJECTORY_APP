Projet Kepler

1. Attendus du projet :

- Réaliser un projet permettant de calculer la trajectoire des planètes du système solaire, dans un réferentiel solaire
- Création de nombreuses fonctions en Langage C et leurs explications pour décrire la trajectoire des planètes, exprimer par des vecteurs de position et de vitesse.
- Création de fonctions de calculs en Langage C pour nous aider à déterminer la trajectoire.
- Modèle d'Euler et le modèle d'Euler asymétrique seront à implémenter.
- Réalisation d'une partie énergétique. Elle sera utile à la vérification des calculs (l'énergie des planètes doit se conserver au cours du temps).
- Affichage de la trajectoire cacluclée avec l'exportation des données dans un fichier JSON.
- Création d'une interface graphique avec de nombreuses pages.

3. Engagement sur la liste des fonctionnalités

Nécessaire :

Création des structures :
- Déclaration des structures (Vector, Point, Path, Planet, List) avec une bonne organisation, soit diviser dans les fichier ".c" et ".h".

Codage de fonctions de création, d'accès et d'opérations sur les structures :
Les fonctions qui vont être créer peuvent être répartis sous différetes catégories :
- Fonctions de création et suppression : Permet d'initialiser la structure et de la supprimer en libérant la mémoire dynamique.
- Fonction d'affichage de la structure : Permet d'afficher les différentes données stockées dans la structure.
- Fonction d'accès à la structure : Permet d'accéder aux données de la structure sans perturber le principe d'encapsulation.
- Fonction d'opérations : Permet d'effectuer différentes opérations.
- Fonctions de test de la structure : Permet de créer une fonction sans paramètres dans le main afin de tester la structure.

Codages des fonctions de calculs :
Fonctions qui vont aider aux calculs de la méthode d'Euler.
- Fonction de calcul d'accélération à un instant t : Fonction qui permet de simplifier le code et de calculer l'accélération à un instant t.
- Fonction pour la méthode d'Euler : Appliquer l'algorithme de la méthode d'Euler pour obtenir la trajectoire.

Extraction des données dans un ficher JSON :
- Fonctions d'extraction : Fonction qui permet de prendre une planète, qui extrait et écrit sa trajectoire dans un fichier JSON.

Vérifications énergétiques :
- Fonctions de verifications énergétique : Fonction qui vérifie si l'énergie est bien constante au cours du temps(avec marge d'erreur).

Création d'une page web :
- Imporation des données JSON : Fonction qui importe les données et les utilises.

- Création de l'interface web initiale, elle comportera donc plusieurs pages :

  1. Acceuil : Présentation du projet.
  2. Présentation : C'est ici qu'il y a aura le Canvas avec la trajectoire affichée.
  3. Informations : Il y aura donc la documentations astronomique sur chaque planète du système solaire, avec une page pour le lexique.
  4. Sources : On y retrouvera donc toutes les sources utilisées.
  5. Trajectoire : On pourra voir les réels trajectoires de chaque planète(on pourra notamment voir le décallage).

Recommandé :

Amélioration du calcul à l'aide de la méthode d'Euler asymétrique :
- Fonctions de calcul amélioré : Fonction qui calcul la trajectoire à l'aide de la méthode d'Euler asymétrique.

Calculer la trajectoire de toutes les planètes :
- Fonction addPlanet : Fonction qui automatisera le processus de calcul à toutes la planètes, ainsi, on pourra exporter en JSON la trajetcoire de toutes les planètes que l'on souhaite.

Ambitieux :

Obtenir une interface graphique en C :
- Menu : Nous allons coder un menu en C, afin que l'utilisateur puisse effectuer certaines actions

Amélioration de l'interface web :
- Ajouter des boutons à l'interface Web : 2 boutons start et stop pour contrôler le Canvas. - Ajouter un menu : Permet de choisir quelle trajectoire on veut afficher ou non.
Amélioration des calculs :
- Modifier les fonctions de calculs en ajoutant les forces des autres planètes sur la planète dont on calcule la trajectoire.

En plus :
